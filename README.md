GCM-via-PHP
===========

Send GCM Messages via PHP and handle the Result-Codes. Super lightweight!

Module Requirements:
----
* php5-curl

Usage Example
----
Add it to your composer-dependencies like this:
```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/Jan1337z/GCM-via-PHP"
    }
],
```
